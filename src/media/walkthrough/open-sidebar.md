# Open the Shofer Sidebar

Shofer lives in the VS Code Activity Bar on the left side of your window. Click the Shofer icon to open the sidebar.

## What You'll See

- **Chat view** — Type messages and interact with the AI
- **Task Selector** — Switch between multiple parallel tasks
- **Mode Selector** — Choose from Code, Architect, Ask, Debug, or Orchestrator
- **File Changes Panel** — Review, accept, or revert every file Shofer modifies

Click the **Open Shofer** button in the step description to get started.
