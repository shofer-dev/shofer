# Start Your First Task

## How It Works

1. Type a message in the chat input bar
2. Shofer reads your workspace files, searches code, runs commands, and edits files
3. Review changes in the **File Changes Panel** — accept or revert each one
4. The **Context Window Bar** shows the current token usage and cost of the task

## Your First Prompt

Try the built-in `/init` slash command — it analyzes your codebase and creates an `AGENTS.md` file with build commands, code style, and project conventions. This helps Shofer (and other AI assistants) be immediately productive in your project.

You can also type any plain request — Shofer will explore your workspace and get to work.
