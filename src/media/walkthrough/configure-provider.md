# Configure an AI Provider

Shofer works with virtually any AI model. Choose from Anthropic, OpenAI, Google Gemini, OpenRouter, Ollama (local), and many more.

## Quick Setup

1. Click **Open Settings** in the step description
2. Navigate to the **Providers** tab
3. Click **New Profile** and choose your provider
4. Enter your API key (stored securely in your OS credential store)
5. Select a model from the dropdown

## Model Recommendations

- **Claude Sonnet 4.5** — best all-around coding performance
- **GPT-5.2 Codex** — strong code generation
- **Gemini 2.5 Flash** — large context window, cost-effective
- **Ollama with local models** — fully offline, no API costs

You can create multiple profiles and switch between them anytime via the **API Config Selector** in the chat input bar.
