# Welcome to Shofer

Shofer is a new (June 2026) open-source AI coding agent for VS Code with unparalleled parallelism, usability and observability. Specify multi-agent workflows declaratively, and watch them execute as live diagrams — on top of all the standard features you expect from your AI-powered development environment.

## What You'll Learn

- **Open Shofer** — Access the sidebar and chat interface
- **Choose a Mode** — Pick the right mode for your task (Code, Architect, Debug, Code Search, Web Search, Reviewer)
- **Configure a Provider** — Connect one of the many AI model providers that Shofer supports
- **Start a Task** — Type a message and let Shofer work
- **Explore Settings** — Customize auto-approval, cost limits, MCP servers, and more
- **Learn More** — Documentation, user manual, and community

## Optional Next Steps

Once you're comfortable, explore these powerful features:

- **Git Worktrees** — Run parallel tasks on different branches in one window
- **RAG Indexing** — Semantic code and git log search for lightning-fast context gathering
- **Live Memory** — A persistent AI companion that accumulates codebase knowledge

Let's get started!
