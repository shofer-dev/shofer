# Welcome to Shofer

Shofer is a **state-of-the-art open-source AI coding assistant** that runs inside VS Code. It can generate, refactor, and debug code across your entire workspace — powered by the AI model of your choice.

## What You'll Learn

- **Open Shofer** — Access the sidebar and chat interface
- **Choose a Mode** — Pick the right mode for your task (Code, Architect, Ask, Debug, Orchestrator)
- **Configure a Provider** — Connect an AI model (Anthropic, OpenAI, OpenRouter, or local)
- **Start a Task** — Type a message and let Shofer work
- **Explore Settings** — Customize auto-approval, cost limits, MCP servers, and more
- **Learn More** — Documentation, user manual, and community

Let's get started!
